# ИИ Маркетолог | Landing Page

Лендинг для продвижения услуги ИИ-маркетолога для выхода на стабильные продажи 300-500к в месяц.

## Описание

Одностраничный сайт с современным минималистичным дизайном, плавными анимациями и адаптивной вёрсткой.

## Структура

- **Hero** — главный экран с заголовком и CTA
- **Кому подходит** — три колонки с целевыми аудиториями
- **Как работает** — 4 шага от анализа до запуска продаж
- **Блок с цифрой** — акцент на 300-500к стабильных продаж
- **Призыв к действию** — финальная кнопка

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- CSS Animations
- SVG Favicon (emoji)
